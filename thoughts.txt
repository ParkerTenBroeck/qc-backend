the first qc1 script launches the page with autofilled oem_serial, make_model and build_type.

once the qc1 sheet is ready for the next step save will be pressed and the current form will be saved to the database.
a single file called qc_ident will be saved to the downloads folder of the qced computer.

when qc2 is ready a second script will be ran qc2. this script will look for the file qc_ident.
if found it will load the corresponding form in edit mode. otherwise it will prompt qc2 to manually search for the computer(through whatever means).

once the qc page is loaded for the second time the identifer will be deleted from the downloads folder.
